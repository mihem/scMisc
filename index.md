# scMisc

`scMisc` is an R package that provides helper functions for scRNA-seq
analysis with a focus on Seurat.

# Installation

To install the package, use:

``` r
remotes::install_github("mihem/scMisc")
```

# Usage

To find out how to use the package, please refer to the
[vignette](https://mihem.github.io/scMisc/).
